# Marketing — Workspace

Riccardo's marketing workspace. Each campaign, initiative, or content project lives in its own folder.

## Structure Rules

- **Every campaign/initiative gets its own folder** — no loose files at the root
- Folder name should be descriptive and kebab-case (e.g. `q2-email-campaign/`)
- Each project folder should have its own context (copy, assets, analytics) self-contained within it
- `plans/` and `docs/` at root are for cross-project planning and documentation

## Available Tools

This workspace has full access to marketing agents (content, campaigns, SEO, social, analytics) and MCP integrations (Gmail, Slack, Calendar, Canva, Gamma, ElevenLabs, n8n). See `CLAUDE.md` for details.

## Adding a New Project

1. Create a new folder: `Marketing/your-campaign-name/`
2. Start building inside it — keep everything contained there
3. Update this README with a short entry under Projects
